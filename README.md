# PHP Session Login Project

مشروع بسيط بلغة **PHP** يوضح كيفية **تسجيل مستخدم، حفظه في Session، وعرض بياناته** مع رسائل نجاح عند تسجيل الدخول والخروج.

---

## **ميزات المشروع**
- تسجيل مستخدم باسم وبريد إلكتروني.
- حفظ بيانات المستخدم مؤقتًا باستخدام **Session**.
- صفحة Dashboard تعرض بيانات المستخدم.
- حماية الصفحات من الوصول بدون تسجيل.
- عرض رسالة عند تسجيل الدخول: "تم تسجيل الدخول بنجاح!"
- عرض رسالة عند تسجيل الخروج: "تم تسجيل الخروج بنجاح!"
- تصميم بسيط وجذاب باستخدام CSS.
- تجربة مستخدم سهلة وواضحة.

---

## **ملفات المشروع**
| الملف | الوصف |
|-------|-------|
| `index.php` | صفحة تسجيل الدخول / التسجيل |
| `dashboard.php` | صفحة عرض بيانات المستخدم |
| `logout.php` | تسجيل الخروج وحذف بيانات Session |
